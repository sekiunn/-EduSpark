\# Enterprise Anti-AI Commercial UI System

企业级去AI塑料感商用UI设计系统 | 新建+重构双引擎

\## 核心能力

1\. 从零生成：企业官网/SaaS平台/中后台管理系统 商业级无AI感页面

2\. 存量优化：一键重构AI生成页面，消除模板感/塑料感，保留核心业务逻辑

3\. 工程化落地：符合大厂前端规范，支持SEO优化、响应式适配、无障碍合规、多端兼容

\## 禁用规则（企业级强制）

1\. 抛弃iframe架构、零代码原型架构，采用纯企业级前端工程标准

2\. 禁止AI直接定稿，所有输出必须经过人工校准（商业规范+合规校验）

3\. 禁止复古风格、过度装饰，坚守商用极简、高级质感原则
